# Haythem Photo Hub 📸

A professional photography portfolio site for Haythem, featuring multi-language support (English, French, and Tunisian Arabic) and integrated social media content.

## ✨ Features

- **Multi-language Support**: Seamlessly switch between English, French, and Tunisian Arabic (Derja) with full RTL support.
- **Cinema Mode Reels**: Integrated Instagram Reels with a custom "Cinema Crop" for a premium, dark-themed viewing experience.
- **Localized Portfolio**: Custom branding and high-quality image galleries.
- **Social Integration**: Direct links to Instagram, Facebook, and WhatsApp.

## 🚀 Tech Stack

- **Frontend**: React (Vite) + Tailwind CSS + Framer Motion
- **Internationalization**: Custom `LanguageProvider` with RTL logic.
- **Tools**: PNPM Workspaces (Monorepo)

## 🛠️ Local Development

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Run the photography frontend**:
   ```bash
   pnpm --filter @workspace/haythem-photography run dev
   ```

3. **URL**: `http://localhost:5000`
